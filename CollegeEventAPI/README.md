# College Event Management System

A web application built using Flask and REST APIs to manage college events.

## Features

- Add New Events
- View Events
- REST API (CRUD)
- Frontend using HTML, CSS & JavaScript
- API Testing using Postman

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Postman

## Project Structure

```
CollegeEventAPI/
│
├── app.py
├── templates/
│   └── index.html
└── static/
    ├── style.css
    └── script.js
```

## API Endpoints

- GET /events
- POST /events
- PUT /events/<id>
- DELETE /events/<id>
